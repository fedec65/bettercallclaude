- `search_decisions`: Search BGE/ATF/DTF by keywords, statute articles, legal domain, and date range.
- `get_decision_by_citation`: Retrieve a specific decision when a citation is provided or discovered.

Search broadly first, then narrow by relevance. Retrieve at least the 3 most relevant precedents.

### Step 2 -- Statutory Analysis

Apply Swiss interpretation methodology in this order:

1. **Grammatical** (Wortlaut / texte / testo): Start with the ordinary meaning of the statutory text. Check all three official language versions for consistency.
2. **Systematic** (Systematik / systematique / sistematica): Consider the provision's position within the statute and the broader legal system.
3. **Teleological** (Zweck / but / scopo): Identify the legislative purpose using travaux preparatoires where available.
4. **Historical** (Entstehungsgeschichte / historique / storica): Trace the provision's development over time.
